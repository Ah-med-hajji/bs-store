# BS Store

**Modern menswear landing page — smooth scroll, GSAP animations, and Supabase integration.**

[![MIT License](https://img.shields.io/badge/License-MIT-0A0A0A?style=for-the-badge&logo=opensourceinitiative&logoColor=white)](LICENSE)

</div>

---

## 🎯 Problem It Solves

Tunisian fashion brands need fast, visually striking landing pages that showcase seasonal collections and drive conversions — without paying for expensive platforms.

---

## ✨ Features

- Smooth scroll experience with Lenis
- GSAP-powered page transitions and animations
- Supabase backend for product data and auth
- SEO-ready with sitemap and robots.txt
- Fully responsive across all devices

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/Ah-med-hajji/bs-store.git
cd bs-store
npm install
```

### Environment Variables

```bash
cp .env.example .env
# Add your Supabase URL and anon key
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
